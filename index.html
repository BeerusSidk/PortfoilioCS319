Projects

Project 1

My first project is called Scheduler, an app for ISU students, which takes courses the students want to take or parse their degree audit and then make a schedule. which is written in SpringBoot for Backend, and Android for Frontend. It takes the input from the students to generate the schedule for them. They can either manually answer some questions, or import the PDF degree audit.

    I was responsible for the Backend:

  • Implementing user authentication using JWT (JSON Web Token) for secure user verification.
  • Developing PDF parsing algorithms to extract and interpret data from degree audits
  • Designing and implementing databases for Users and Courses for efficient data management and retrieval.
  • Setting up Swagger documentation for the application's API, enhancing transparency and ease of use for developers.
